My List of Movies!

The purpose of this API is to recieve information on Movies, Directors, and genres in order to learn more about movies I have watched or am interested in.

Request URL HTTP Method Request Body Data Format Response Body Data Format
Get a List of ALL movies /movies GET None A JSON object holding an array with all movies
Get data about a single movie by title /movies/:title GET None A JSON object holding data about a single movie, including a description, the genre and the director
Get data about a genre by name /movies/:genre/:genreName GET None A JSON object holding data about a genre including the name and a description.
Get data about a director by name movies/:directors/directorName GET None A JSON object holding data about a single director including their name and a short biography
Register a new user /users POST A JSON object holding user data A JSON object holding user data including ID
Update username /users/:id PUT A JSON obect holding user data A JSON object holding user data and ID
Add movie to user's favorite movies /users/:id/:movieTitle POST None A Text message indicating success or failure of adding movie to users favorites
Remove a movie from user's favorite movies /users/:id/:movieTitle DELETE None A Text message indicating success or failure of removing movie from users favorites
Delete user registration /users/:id DELETE None A Text message indicating success or failure or deleteing user
